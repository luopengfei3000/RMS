## BM.Tech
    基于 SSM+Shiro+Layui+Easyui 的后台管理系统（maven版）

## 介绍
    BM 致力于更完善,代码注释更丰富,更易于理解学习上手的后台管理系统
## 技术 
    SSM 框架  ,  Shiro 权限管理  , Layui + Easyui 页面技术 
## 功能
    1.用户信息管理
    
    2.角色管理

    3.资源管理

    4.部门资源

    5.字典管理

    6.日志管理
      
    后期功能陆续更新(boot版等等)
